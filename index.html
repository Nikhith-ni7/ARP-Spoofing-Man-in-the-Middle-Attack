05
Mitigations
DEF-01
Dynamic ARP Inspection
Validates ARP packets against DHCP snooping table on managed switches. Blocks spoofed ARP replies at the network layer.
DEF-02
Static ARP Entries
Pins critical hosts to their real MAC addresses. Cannot be overwritten by gratuitous ARP replies.
DEF-03
Encrypted DNS (DoH/DoT)
Encrypts DNS queries so an active MITM cannot read which domains the victim is resolving.
DEF-04
VPN / End-to-End Encryption
Encrypts all traffic so intercepted packets are unreadable regardless of MITM position.
DEF-05
IDS / Network Monitoring
Monitors for duplicate ARP alerts, anomalous MAC changes, and unusual traffic patterns in real time.
